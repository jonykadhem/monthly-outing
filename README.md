# Monthly-Outing
A restaurant review web app that helps a group of reviewers manage the restaurants they've visited and discover new ones to try, including:
- browsing reviewed restaurants and their ratings
- suggesting new restaurants to be visited
- rating restaurants by price, taste, service, and quantity
- managing user roles through an admin panel
